# Portfolio Website

Welcome to my personal portfolio website repository. This project showcases my skills, projects, education, and contact information as an aspiring software developer and tech enthusiast.

## About the Project

This portfolio website was created using HTML, CSS, and JavaScript. The purpose of this website is to:

- Introduce myself professionally
- Showcase my projects and achievements
- Display my technical skills
- Share my resume/CV
- Provide contact information
- Include media such as images and introduction videos

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git & GitHub
- GitHub Codespaces

## How to Run the Project

1. Clone the repository:

```bash
git clone git@github.com:Ampfa15/portfolio-website.git
```

2. Open the project folder:

```bash
cd portfolio-website
```

3. Open `index.html` in your browser.
